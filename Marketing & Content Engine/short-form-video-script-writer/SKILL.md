# Short-Form Video Script Writer

Turn a rough idea into a short-form video script engineered to stop the scroll and retain attention.

## Core workflow

When the user provides a topic, idea, offer, article, product, story, lesson, or campaign:

1. Identify:
   - target audience
   - platform
   - video goal
   - viewer pain or desire
   - main takeaway
   - offer or CTA
   - tone
   - desired length
   - visual format if provided

2. If details are missing, make practical assumptions and label them briefly.

3. Create a script with a strong first 3 seconds.

4. Build retention through pattern interrupts, open loops, specificity, fast pacing, and clear payoff.

5. End with a natural CTA that fits the video goal.

6. Use `references/short-form-retention-framework.md` when the user asks for multiple variants, hooks only, storyboards, retention edits, or platform-specific scripting.

Use this format unless the user requests another format:


## First 3 seconds

The opening must immediately create one of these:

- curiosity
- tension
- relevance
- surprise
- pain recognition
- clear promised payoff
- identity match
- contrarian insight

Good hook patterns:

- "Most [audience] get [topic] wrong because..."
- "If you're [audience], stop doing this..."
- "The fastest way to [outcome] is not [common tactic]."
- "I would not [action] until you understand this."
- "Here is the mistake costing [audience] [business consequence]."
- "You do not need [common belief] to get [outcome]."

Avoid weak hooks:

- "Hey guys"
- "Today I want to talk about..."
- "Welcome back"
- "Here are some tips"
- "In this video, I will explain..."

## Script principles

Every script should:

- start in the middle of the viewer's problem
- make the payoff clear early
- use short spoken lines
- change visual beats every few seconds
- avoid long setup
- avoid generic advice
- include one main idea
- end with a clear next step

- service promotion

## Script types

Choose the best type unless the user specifies:

### Educational
Teach one useful idea quickly.

### Contrarian
Challenge a common belief and replace it with a better one.

### Problem-solution
Name a problem, explain why it happens, show a better path.

### Story
Use a quick narrative with a lesson or business insight.

### List
Deliver 3-5 points quickly with strong pacing.

### Offer-led
Connect a pain point to a product, service, event, or lead magnet.

### Proof-led
Use a result, case study, or observation as the hook, without inventing proof.

## Platform adaptation

### Instagram Reels
Punchy, visual, conversational, fast overlay text.

### TikTok
More casual, immediate, direct, strong pattern interrupts.

### YouTube Shorts
Clear payoff, strong structure, high retention, searchable topic alignment.

### LinkedIn video
Professional, insight-led, business outcome-focused, less slang.

### Paid social video
Offer clarity, pain, proof, and CTA need to appear earlier.

## Optional variants

When useful, provide:

- 5 hook options
- 3 script angles
- a 15-second version
- a 30-second version
- a 60-second version
- storyboard notes
- B-roll list
- caption variants
- CTA variants

## Final quality check

Before responding, verify that:

- the first line works without context
- the first 3 seconds give a reason to keep watching
- the script has one clear idea
- each beat earns attention
- the CTA is natural and specific
- the script is speakable out loud
- claims are not invented
- the output is ready to record
