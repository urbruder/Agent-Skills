# Tone Patterns Reference — Inbox Triage

Load this file when the email context requires special tone handling.

---

## Conflict & Complaint Emails

**Signs:** Caps, exclamation marks, "unacceptable", "disappointed", "escalate", "legal".

**Drafting rules:**
- Open by acknowledging their frustration (one sentence — don't grovel)
- Don't match their energy. Be calm, clear, and specific
- Never use "I understand how you feel" — it reads as patronising
- State what *will* happen, not what you "hope" or "intend"
- If you're at fault: acknowledge it briefly and move to resolution fast
- If you're *not* at fault: don't be defensive; just be factual and offer a path forward

**Template structure:**
```
[Acknowledge the issue in one sentence]
[What happened / what the situation actually is — factual, no spin]
[What you're doing / what happens next — concrete]
[Any ask from them if needed]
[Neutral, professional close]
```

**Example opener (complaint):**
> "Thanks for flagging this — I can see why that was frustrating."
*(not: "I'm so sorry you feel that way" / "We apologize for any inconvenience")*

## Senior Executive / Board-Level Contact

**Signs:** C-suite titles, board members, investors, regulatory bodies.

**Drafting rules:**
- Lead with the bottom line — they don't read setup paragraphs
- One ask per email maximum
- No bullet lists longer than 3 items; if you need more, attach a doc
- Remove every hedge: "I think", "perhaps", "it might be worth considering" → state directly
- Time-stamp any commitments: "by Friday EOD" not "soon"
- Sign off with your full name and title if available

**Template structure:**
```
[Bottom line / answer / decision — first sentence]
[One paragraph of essential context only]
[Single clear ask or next step, if any]
[Sign-off]
```

## De-escalation (Internal Team Conflict)

**Signs:** Passive-aggressive language, complaints about a colleague, venting, "not my fault".

**Drafting rules:**
- Acknowledge the difficulty without taking sides
- Don't pile on about whoever they're complaining about — stay neutral
- Redirect toward the solution, not the problem
- Suggest a call/meeting for anything that's gone 2+ email rounds
- If the issue is serious (harassment, discrimination): draft very conservatively and add
  the HR flag below

**Flag to add:**
> ⚠️ *This thread touches on workplace relations. Consider looping in HR or a manager before responding.*

---

## HR / Legal / Financial Sensitivity

**Trigger words:** termination, lawsuit, GDPR, settlement, salary, PIP, investigation,
NDA, equity, medical leave, disability.

**Drafting rules:**
- Write shorter than normal — every word is potentially quoted
- Avoid admissions of fault or commitments that haven't been approved
- Don't express personal opinions about the situation
- Stick to facts and process

**Always add this note:**
> ⚠️ *This email touches on [legal/HR/financial] matters. Recommend having [legal/HR/your advisor] review before sending.*

## Declining Requests

**Types:** Saying no to a meeting, project, collaboration, discount, reference, intro.

**Structure (the "warm no"):**
```
[Acknowledge what they asked for — one sentence]
[Brief honest reason — don't over-explain]
[Alternative or offer of limited help, if genuine]
[Warm close]
```

**Rules:**
- Don't apologise more than once
- Don't say "unfortunately" — it's filler
- Don't invent an excuse — a short honest reason respects them more
- If offering an alternative, make it real. Don't say "happy to connect sometime" if you won't follow up.

**Example:**
> "Thanks for thinking of me for this — I won't be able to take it on right now given current commitments.
> If you're still looking in Q3, I'd be glad to revisit."

## Follow-Up / Chase Emails

**When used:** User is chasing someone who hasn't replied.

**Rules:**
- One short sentence acknowledging the previous message
- Restate the ask clearly (don't make them scroll)
- Specific deadline if one exists
- Never say "just following up" as the opener — it's weak

**Template:**
```
[Reference previous message — one sentence]
[Restated ask — clear, one sentence]
[Deadline or "by end of week" if applicable]
```

**Example opener:**
> "Wanted to make sure my last message didn't get buried — we need your sign-off on [X] by Thursday to stay on schedule."

---

## Non-English Threads

**Rules:**
- Match the language of the original email exactly
- Match formality level — many languages have formal/informal registers that differ significantly
  from English (German Sie/du, French vous/tu, Spanish usted/tú)
- Default to formal if unclear — easier to warm up than to walk back
- Don't translate proper nouns (company names, product names, people's names)
- Flag to the user: *"Draft written in [language] — please review for local tone conventions."*

## Meeting Request Responses

**Accepting:**
> "[Name], confirmed for [day] at [time]. [Video link / location] works for me.
> Looking forward to it."

**Proposing alternate time:**
> "Thanks — that slot doesn't work on my end. Would [Day, Time] or [Day, Time] suit you instead?"

**Declining and offering alternative format:**
> "I won't be able to meet this month, but happy to answer your questions over email if that works."

## Cold Outreach (Responding to it)

**When to draft a reply:** User got a cold email and wants to respond (not ignore it).

**Types:**
- Sales pitch → polite decline or pass to the right person
- Journalist / press inquiry → cautious, route to comms if exists
- Job applicant → warm acknowledgment, clear timeline
- Partnership / collab → gauge interest, set up a call

**Default structure:**
```
[One-line acknowledgment of what they sent]
[Direct answer: interested / not interested / not the right person]
[Next step if interested, or graceful close if not]
```
