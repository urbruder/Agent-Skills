# Return Analysis Framework

## Common return clusters

- wrong size or fit
- not as expected
- quality issue
- damaged in transit
- late delivery
- wrong item
- changed mind
- color/material mismatch
- compatibility issue
- difficult to use
- missing parts
- duplicate purchase
- price/value disappointment

## Root-cause categories

### Expectation mismatch
Fix product page, images, videos, sizing charts, copy, FAQs.

### Product issue
Fix materials, quality control, design, packaging, supplier.

### Operational issue
Fix fulfillment, shipping, picking, packing, support.

### Buyer fit issue
Fix targeting, segmentation, recommendations, eligibility, use-case clarity.

## Prioritization

Score fixes by:

- frequency
- revenue impact
- margin impact
- preventability
- implementation effort
- customer trust impact

## Useful outputs

- product page changes
- image shot list
- FAQ updates
- sizing guidance
- packaging improvements
- supplier questions
- QA checklist
- support macros
- post-purchase education emails
