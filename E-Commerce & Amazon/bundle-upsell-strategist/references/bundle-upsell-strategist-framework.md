# Bundle Strategy Framework

## Bundle types

- starter kit
- routine bundle
- replenish bundle
- problem-solution bundle
- gift bundle
- premium upgrade
- travel/sample bundle
- family/multi-pack
- seasonal bundle
- subscribe-and-save bundle

## Pairing logic

Strong bundles pair by:

- same use occasion
- before/after sequence
- accessory need
- replenishment timing
- giftability
- routine completion
- customer segment
- problem solved

## Upsell placements

- product page
- cart drawer
- checkout
- post-purchase
- thank-you page
- email/SMS
- subscription portal

## Metrics

Track AOV, attach rate, conversion rate, margin, refund rate, subscription adoption, and repeat purchase.
